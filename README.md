# Nizam & Nihala – Wedding Invitation

A beautiful, responsive single-page digital wedding reception invitation for Nizam Nazar and Nihala Shakeer.

## Features
- **Elegant Aesthetic**: Curated color palette, elegant typography, and subtle falling petal animations.
- **Countdown Timer**: Real-time countdown to the wedding reception (July 13, 2026).
- **Add to Calendar**: Interactive options to add the event directly to Google Calendar or download an `.ics` file for Outlook/Apple Calendar.
- **Get Directions**: One-click navigation link to the reception venue (Mattool Centre) on Google Maps.

## How to Run Locally
Open `index.html` directly in your browser, or serve it using Python:
```bash
python -m http.server 8000
```
Then visit `http://localhost:8000`.
